# Quotes of Day App

This is a simple Quotes of day app This is a web application designed to provide you with a daily dose of inspiration. Using the Quotable API

## Features

- Listen to Quotes.
- Copy Quotes.
- Share on Twitter.
- Responsive Design.

## Technologies Used

- HTML 
- CSS 
- Bootstrap 
- Font Awesome
- JavaScript 
- Quotable API (https://github.com/lukePeavey/quotable)
